# Ahnaf's Personal Website

Welcome to Ahnaf's personal website! This project showcases the skills and profile of a student named Ahnaf. The website is built using Bootstrap for responsive design and includes colorful icons and emojis to represent Ahnaf's skills.

## Project Structure

The project consists of the following files and directories:

```
ahnaf-website
├── index.html        # Main HTML document for the website
├── css
│   └── styles.css    # Custom CSS styles
├── js
│   └── scripts.js     # JavaScript functionality
├── assets
│   ├── icons          # Colorful icons for skills
│   └── fonts          # Custom fonts for typography
├── .gitignore         # Files and directories to ignore by Git
└── README.md          # Documentation for the project
```

## Setup Instructions

1. **Clone the Repository**: 
   Clone this repository to your local machine using:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory**:
   ```
   cd ahnaf-website
   ```

3. **Open `index.html`**:
   Open `index.html` in your preferred web browser to view the website.

4. **Customize Styles**:
   Modify `css/styles.css` to change the appearance of the website as needed.

5. **Add Functionality**:
   Use `js/scripts.js` to add any interactive elements or animations.

## Icons and Fonts

- Icons are stored in the `assets/icons` directory. You can use libraries like Font Awesome or custom images.
- Custom fonts can be added to the `assets/fonts` directory to enhance typography.
